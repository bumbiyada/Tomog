Руководство пользователя

Заходя на сайт впервые, пользователь попадает на главную страницу. Сверху находится навигационная панель с кнопками "главная", таблицы (с раскрывающимся списком кнопок), "Термины", "Технологии", "О проекте", "Авторизация". В центре находится большая кнопка для перехода на главный раздел сайта "Томографы".

В списке таблиц представлены такие разделы: "томографы", "производители", "трубки", "генераторы","столы" Каждый из этих разделов представляет собой таблицу с характеристиками этих предметов. В разделе "томографы" из-за большого количества характеристик, изначально отображается часть характеристик, для отображения всех характеристик выбранного томографа, в таблице, справа имеется кнопка "подробнее". В разделе томографы также имеется фильтр по таким позициям: "производитель", "количество трубок", "количество срезов", "страна производства".

В разделе "Термины" представлены краткие определения слов, которые могут быть неизвестны пользователю

В разделе "Технологии" в виде списка представлены технологии, применяемые производителями в томографах для улучшения качества изображений, избавления от артефактов, уменьшения дозы облучения и тд.

В разделе "О проекте" представлена информация об используемом програмном обеспечении

Нажав на кнопку "Авторизация" и введя нужные для авторизации данные на странице "Login", пользователь попадает на страницу административного раздела. Он создан для добавления, редактирования и удаления записей из базы данных.
